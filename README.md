# script-monitoramento-dilata
Este script em Python automatiza o monitoramento de uma lista de espera no Feegow usando Selenium. Ele faz login, rastreia o botão "Chamar senha" e verifica um segundo botão "DILATA". Se "DILATA" for encontrado, um alerta é tocado por voz sintetizada. O script também fecha modais automaticamente e roda em modo headless para maior eficiência.
